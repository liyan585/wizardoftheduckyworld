# Wizard of the Ducky World

**Discover your digital personality in 10 questions.**

[Play Now](https://wizardoftheduckyworld.com)

## What is it?

An interactive personality quiz that reveals your digital identity. Answer 10 questions and get a unique personality card based on the 16 personality types — but make it internet culture.

## How it works

1. Enter the game
2. Answer 10 questions
3. Get your personality card
4. Share it

## Stack

- Next.js 15 · React · TypeScript · CSS Modules

## Quick Start
```bash
npm install
npm run dev
```
